План для первого шага в IT

Мы понимаем, что новичку сложно сделать первый шаг в IT. У нас есть простой путь с видеоинструкцией и разбором типичных ошибок, который можно пройти за один вечер и получить первый рабочий сайт.

фрагмент инструкции

Один из шагов внутри видео и текста: 

<step id="1">
  открыть index.html
  заменить текст на свой
  сохранить файл
</step>
Как проходит вечер

План из трёх шагов

  1. 1
    Открыть материалы

    Получаешь доступ к шаблону, видео и инструкции по публикации.

  2. 2
    Повторить шаги по видео

    Запускаешь видео и повторяешь действия: где открыть файл, что заменить, что сохранить.

  3. 3
    Опубликовать сайт

    Проходишь по инструкции публикации на GitHub Pages и получаешь ссылку на свой сайт.
