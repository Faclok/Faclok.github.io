Технические задания

Два уровня сложности: 750₽ — базовый лендинг. 1900₽ — расширенный продукт с анимациями и сложными сценариями.

ТЗ для 750₽

Страница-результат + три шаблона. Чистая структура, минимум сложностей — можно сверстать за один вечер.

уровень: базовый 1 страница + 3 шаблона

Продвинутое оформление, motion-эффекты, акцент на эмоции и динамику. Подходит под более «дорогой» опыт.

уровень: продвинутый сложные сценарии

Что получает человек после покупки

Каждый шаблон — это полный HTML и CSS, который ученик сможет отредактировать. Дизайн минималистичный, логика прозрачная.

Шаблоны пакета 750₽

  • template_1.html — личное резюме Стартовый сайт-квиз о человеке
  • template_2.html — результат за вечер Фокус на демонстрацию результата
  • template_3.html — базовый лендинг Чистая структура блоков

Шаблоны пакета 1900₽

  • template_1.html — storytelling-лендинг История, которая ведёт к офферу
  • template_2.html — сайт с анимациями по скроллу Поэтапные сцены и переходы
  • template_3.html — сложный CSS + JS Максимум возможностей для «поиграться»
